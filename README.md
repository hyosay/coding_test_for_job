## find, count, strip
```python

numStr = '1231212'
print(numStr.find('23', 1)

# strip
strA = '  abc  b '
print(strA.rstrip()) # 우공백 제거
print(strA.lstrip()) # 좌공백 제거
print(strA.strip()) # 좌우 공백 제거
```
